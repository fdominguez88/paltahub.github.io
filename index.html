How Palta Hub Works

Route

  • SDR selects prospect and AE (or system auto-assigns).
  • System checks AE's availability via calendar.
  • SDR confirms meeting time.
  • System books meeting, updates CRM, and sends notifications.

Stream

  • Prospect submits a form on your website.
  • System qualifies the lead based on predefined rules.
  • Scheduling widget appears with available times.
  • Prospect selects a time, and system books the meeting.

Lane

  • Sales Manager defines routing rules (territory, industry, deal size).
  • System auto-assigns leads based on these rules.
  • Assigned rep receives instant notification.
